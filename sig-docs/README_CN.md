# 文档特别兴趣小组

文档是一个开源项目成功的关键。开源项目在运营的过程中会有不少的文档工作，不同的开发人员写出来的文档会有很大的区别，特别是对外发布的文档。SIG Docs 负责对这些文档进行润色和校对，把文档的质量提升一个档次。

[章程](charter-cn.md) 定义了 Docs 特别兴趣小组的范围和治理。
